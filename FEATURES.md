## 技术架构

### 前端
- Flutter 3.x
- Material Design 3

### 后端
- Supabase（PostgreSQL）
- 实时数据同步

### AI 服务
- DeepSeek API
- 模型：deepseek-v4-pro

### 存储
- 本地：SharedPreferences
- 云端：Supabase
