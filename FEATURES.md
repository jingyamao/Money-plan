## 代码规范

### 文件命名
- 使用 snake_case
- 例如：`transaction_list_screen.dart`

### 类命名
- 使用 PascalCase
- 例如：`TransactionListScreen`

### 常量命名
- 使用 camelCase
- 例如：`primaryColor`

### 注释
- 公共 API 必须添加文档注释
- 复杂逻辑添加行内注释
