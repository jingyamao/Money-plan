# Money Plan 使用指南

## 首页功能

### 1. 快速记账
- 点击底部中间 **+** 按钮，打开完整记账页面
- 点击首页 **"记一笔"** 快捷按钮，弹出快速记账弹窗
- 选择支出/收入 → 输入金额 → 选择分类 → 保存

### 2. 概览卡片
- **今日消费** - 当天所有支出汇总
- **本月剩余** - 月度预算 - 本月已消费
- **当前存款** - 在设置中手动设置
- **月收入** - 在设置中手动设置

### 3. 预算进度
- 显示本月预算使用百分比
- 显示已花费金额和预算总额
- 显示日均可用金额

### 4. AI 洞察
- 点击进入 AI 对话页面
- 可以询问消费分析、理财建议等问题

### 5. 最近交易
- 显示最近 5 笔交易
- 点击可编辑交易
- 左滑可删除交易
- 点击"查看全部"查看所有交易

## 记账功能

### 快速记账（首页）
1. 点击首页 **"记一笔"** 按钮
2. 选择 **支出** 或 **收入**
3. 输入金额
4. 选择分类（餐饮、交通、购物等）
5. 点击 **保存**

### 完整记账（底部 + 按钮）
1. 点击底部中间 **+** 按钮
2. 选择支出/收入
3. 输入金额
4. 选择分类
5. 选择日期（默认今天）
6. 添加备注（可选）
7. 点击 **保存**

### 编辑交易
- 在交易列表中点击任意交易
- 可修改金额、分类、日期、备注
- 点击保存

### 删除交易
- 在交易列表中左滑交易
- 点击删除按钮
- 确认删除

### 搜索交易
- 进入"查看全部"页面
- 点击右上角搜索图标
- 输入关键词搜索

### 筛选交易
- 按类型：全部/支出/收入
- 按日期：选择日期范围

## 预算管理

### 设置月度预算
1. 进入 **设置** → **月度预算**
2. 输入预算金额
3. 保存

### 设置分类预算
1. 进入 **设置** → **分类预算**
2. 选择预算周期（周/月/年）
3. 点击 **添加分类预算**
4. 选择分类，输入预算金额
5. 保存

### 预算提醒
- 消费达到预算 80% 时提醒
- 超出预算时提醒

---

## 存款目标

### 创建目标
1. 进入底部 **目标** 标签
2. 点击右上角 **+** 按钮
3. 输入目标名称（如：买车、旅行）
4. 输入目标金额
5. 选择目标日期（可选）
6. 点击创建

### 目标进度
- 显示已存入金额
- 显示还需存入金额
- 显示预计完成时间

### 完成目标
- 当目标达成时，点击"完成目标"按钮
- 自动从存款中扣除

---

## 固定收支

### 添加固定收支
1. 进入 **设置** → **固定收支**
2. 点击 **+** 按钮
3. 选择支出/收入
4. 输入名称和金额
5. 选择分类
6. 设置每月执行日期
7. 保存

### 用途示例
- 固定支出：房租、水电费、话费
- 固定收入：工资、奖金

---

## 账户管理

### 添加账户
1. 进入 **设置** → **账户管理**
2. 点击 **+** 按钮
3. 选择账户类型（支付宝/微信/现金）
4. 输入账户名称
5. 保存

### 用途
- 记录不同支付方式
- 方便按账户统计消费

## AI 功能

### 进入 AI 对话
- 点击首页 **AI 洞察** 卡片

### 可以问的问题
- "我这个月花了多少钱？"
- "我的钱都花在哪里了？"
- "给我一些省钱建议"
- "按目前速度月底能存多少？"

### 功能特点
- 对话历史自动保存
- 支持 Markdown 格式显示
- 可复制 AI 回答
- 可停止 AI 生成

## 云端同步

### 登录/注册
1. 进入 **设置** → **同步到云端**
2. 输入邮箱和密码
3. 点击登录或注册

### 同步内容
- 交易记录
- 存款目标
- 用户设置（预算、存款、收入）

### 同步方式
- 登录后自动同步
- 添加/修改数据时自动上传

1. 进入 **设置** → **导入账单**
2. 选择 **支付宝账单导入**
3. 按照指南导出 CSV 文件

1. 进入 **设置** → **导入账单**
2. 选择 **微信账单导入**
3. 按照指南导出 CSV 文件
4. 点击开始导入

## 资金联动说明

### 预算联动
```
月度预算 - 本月消费 = 本月剩余
本月剩余 / 剩余天数 = 日均可用
```

### 存款联动
```
月收入 - 固定支出 = 可支配收入
可支配收入 - 生活费预算 = 每月可存款
当前存款 + 每月可存款 × 月数 = 预计存款
```

### 目标联动
```
目标金额 - 当前存款 = 还需存入
还需存入 / 每月可存款 = 预计月数
```

## 常见问题

### Q: 如何修改已记账的交易？
A: 在交易列表中点击该交易，进入编辑页面修改。

### Q: 如何查看某个月的消费？
A: 进入"查看全部"页面，选择日期范围筛选。

### Q: 数据会丢失吗？
A: 登录账号后数据会同步到云端，本地也会保存。

### Q: 如何导出数据？
A: 目前暂不支持导出，后续版本会添加。
